🎮 Number Guessing Game
   A fun and simple Python game where you try to guess the number chosen by the computer.

🚀 How to Play
 - Computer selects a random number between 1 and 100
 - You try to guess the number
 - After each guess, you get hints:
   - 📈 Too high
   - 📉 Too low
 - Try to find the correct number!
 Built With
    - Python 
▶️ How to Run
     python NumberGuessingGame.py
